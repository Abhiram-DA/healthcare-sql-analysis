# Hospital Management SQL Project

This project contains SQL queries solved using a Hospital Management dataset.

## Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions
- JOINs
- Subqueries
- UPDATE Statements
- String Functions

## Objective
The goal of this project was to practice SQL query writing and improve problem-solving skills using healthcare-related data.

## Tools Used
-  MySQL 

## Project Highlights
- Patient analysis
- Physician and nurse insights
- Data filtering and sorting
- Query optimization practice
